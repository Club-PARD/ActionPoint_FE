### 문제
저희는 [팀프로젝트에 익숙하지 않은 대학생들]이 <br>
[회의를 처음부터 끝까지 체계적으로 진행하는 방법을 모르기] 때문에<br>
[구체적인 실행계획을 세우지 않아 결국 프로젝트가 지연되는 문제]를 겪고 있다고 생각합니다.


### 🎯 타겟 🎯
- 의미 없이 끝나는 회의가 반복되어 실행으로 이어지지 않는 것에 답답함을 느끼는 대학생
- 회의를 단순한 대화가 아닌, 진짜 실행으로 연결되는 시간으로 만들고 싶은 대학생
- 회의 때마다 무슨 이야기를 했는지 정리가 안 되고, 결국 "나 뭐 해야 하지?" 하고 방황하는 대학생

**의미없이** 끝나는 회의가 반복되어 실행으로 이어지지 않는 것에 답답함을 느끼는 대학생

### 솔루션

1. 사용자가 작성한 일정을 오늘 날짜 기준으로 확인할 수 있도록 표시합니다.  

2. 사용자가 해야 할 일을 세부적인 단계로 나눌 수 있도록 도와주는 **세분화 기능**을 통해 진입 장벽을 낮춥니다.  

3. 사용자가 원하는 마감 기한을 직접 설정할 수 있습니다.

## ★ 핵심 기능 설명

1. **메인 페이지**: 오늘 등록된 일정을 체크리스트로 확인 가능
2. **캘린더**: 일정 목록을 달력과 당일 기준으로 나누어 확인
3. **일정 등록**:
   - 간단 일정: 제목과 날짜만 입력
   - 세분화 일정: 제목, 시작일, 종료일, 세부 단계별 시간 설정 가능
4. **동기 부여 모달**:
   - 일정 등록 완료 시 응원 메시지 모달
   - 모든 일정 완료 시 축하 모달 제공
